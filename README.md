## 購物車 (With 登入驗證)
**使用 FastAPI + Vue + Vite管理套件 實作練習的購物車系統，單純弄來玩**

## How To Use
### 後端安裝 & 啟動:
```powershell
cd backend
pip install -r requirements.txt
python main.py
```

### 前端安裝 & 啟動:
```powershell
cd frontend
npm install
npm run dev
```

> 本地網頁：http://localhost:5173

> 後端 API：http://localhost:8000

> API 檔案：http://localhost:8000/docs
